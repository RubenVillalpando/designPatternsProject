class LambdaHandlerTemplate

  1. src/models/lambda_handler_template.rb
Superclass: Object

The LambdaHandlerTemplate is an implementation of the Template Method. It allows you to create instances of various ::LambdaHandler classes. Each one with the same logic starting with the ::handle_request, just overriding methods that the implementation will accept.

def handle_request(event)
    method = event['httpMethod']
    case method
    when 'GET'
        handle_get(event)
    when 'POST'
        handle_post(event)
    when 'PUT'
        handle_put(event)
    when 'DELETE'
        handle_delete(event)
    else
        handle_bad_method(method)
    end
end
